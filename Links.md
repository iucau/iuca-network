# Course Server

## Connecting

```bash
ssh -o ServerAliveInterval=5 -o ServerAliveCountMax=1 <login>@iuca.space
```

## Sending Files

```bash
scp <local file> <login>@iuca.space:~/
scp -r <local dir> <login>@iuca.space:~/
```
